## The "Calendar App" (API)

### Showcases:
- CRUDs (ex. update and delete)
- filtering events
- authorization
- user-defined time-zones
- basic feature tests

### Running

Assuming using the DDEV:

- `ddev start`
- `ddev composer install`
- `ddev artisan:migration --seed`
